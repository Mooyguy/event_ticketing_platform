# Event Ticketing Platform

A full-stack web application for browsing events, booking tickets, and generating QR codes.

## Features
- Event browsing
- Ticket booking
- Payment simulation
- QR code generation
- Admin dashboard
- Analytics

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)

## Setup

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
nodemon server.js
